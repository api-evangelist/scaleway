# Scaleway (scaleway)
Scaleway is a European cloud provider offering compute, storage, networking, and AI services. The Scaleway API provides programmatic access to all Scaleway cloud infrastructure and services including Instances, Kubernetes, Generative APIs, serverless containers, object storage, and more.

## APIs

### Scaleway Instance API
The Scaleway Instance API allows you to create, manage, and delete virtual machine instances, manage IP addresses, security groups, and other compute resources.
